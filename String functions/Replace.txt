The REPLACE function replaces all occurrences of a substring with another.

Syntax REPLACE(string, substring1, substring2) The function replaces all occurrences of substring1 in string with substring2. For example, the following code substitutes all occurrences of a dash in the input string with a colon: Click here to view code image SELECT REPLACE('1-a 2-b', '-', ':'); This code returns the output '1:a 2:b'.
